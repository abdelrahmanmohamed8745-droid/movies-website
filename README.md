# Movies Website React

React.js movie browsing website with search, filtering, sorting, watchlist, and detail views.

## Overview

This project is a responsive movie browsing website built with React and Vite. It demonstrates component-based UI development, state management with React Hooks, dynamic filtering, sorting, watchlist interactions, and an interactive details modal.

## Features

- Search movies by title, genre, director, year, or tags
- Filter movies by genre
- Sort movies by rating, year, or title
- Add/remove movies from a watchlist
- View movie details in a modal
- Responsive UI for desktop, tablet, and mobile
- Clean component structure

## Technologies Used

- React.js
- Vite
- JavaScript ES6+
- CSS3
- Lucide React Icons

## How to Run Locally

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Open the local URL shown in the terminal.

## Build for Production

```bash
npm run build
```

## Preview Production Build

